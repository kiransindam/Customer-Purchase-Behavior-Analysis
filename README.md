🛍️ Customer Purchase Behavior Analysis
📌 Project Overview

This project analyzes a retail dataset containing customer demographics, purchasing behavior, product preferences, and transaction details. The goal is to extract meaningful insights that help understand customer buying patterns, product trends, and business decision-making.

# 📂 Dataset Description

The dataset contains the following features:

Column Name	Description
customer_age	Age of the customer
gender	Gender of the customer
item_purchased	Product purchased
category	Product category (Clothing, Footwear, Accessories, etc.)
purchase_amount	Amount spent on the purchase
location	Customer's location (state)
size	Product size (S, M, L, XL)
color	Product color
season	Season of purchase (Winter, Summer, Spring, Fall)
review_rating	Customer rating for the product
subscription_status	Whether the customer is subscribed
shipping_type	Shipping method (Express, Free Shipping, etc.)
discount_applied	Discount usage (Yes/No)
previous_purchases	Number of previous purchases
payment_method	Payment type (Cash, PayPal, Credit Card, etc.)
frequency	Purchase frequency (Weekly, Monthly, etc.)
age_group	Categorized age group
purchase_frequency_days	Days between purchases

# 🎯 Objectives

# 📊 Key Insights
Clothing is the most frequently purchased category
Young adults show higher purchase frequency
Discounts significantly increase purchase volume
Free shipping is the most preferred shipping method
